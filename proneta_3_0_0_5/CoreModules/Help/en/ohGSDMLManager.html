5.3 GSDML Manager

GSDML files (“General Station Description Markup Language”) contain information about the features and capabilities of PROFINET I/O devices, like the number of interfaces or supported baud rates. PRONETA Basic comes already with a large number of GSDML files preinstalled. With the GSDML manager you can include support for further devices, and for devices which were only released after the latest PRONETA Basic version.

Function Bar

On the top of the screen, the following functions are available from the function bar:

Icon

Name

Function

Add GSDML File

Lets you add a single GSD file from disk. Clicking on the icon opens a file browsing dialog.

Add GSDML Folder

Lets you add several GSD files in one action by simply designating a folder where the individual GSD files are stored. Clicking on the icon opens a file browsing dialog.

Download GSDML Files for SIEMENS Devices

Will launch a web browser and establish a connection to the SIEMENS industry online support portal where you can download additional GSD files, e.g. for new devices.
