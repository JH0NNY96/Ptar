4 IO 测试任务

IO 测试任务用于在安装 CPU 之前检查分布式 I/O 设备的接线，并以快速、准确且简单的方式拟定测试结果。

4.1 设备选择模式

在“设备选择”模式下，PRONETA Basic 显示一个符号，表示在当前网络扫描期间发现的所有 SIMATIC ET 200 设备、SIRIUS 电机起动器和 SIPLUS 加热控制系统。

绿色复选标记表示该设备可进行 IO 测试（即该设备的 IP 地址和设备名称有效）。单击要测试的设备以进行选择。在右侧窗口部分，选择设备后会显示一个设备详细信息表（类型、网络组态、端口和模块组态数据等），选中的设备以蓝色背景显示。

要在将设备选作测试备选设备之后继续进行 IO 测试，请双击该设备，或者在模式栏中选择“IO 测试”(IO Test) 选项卡。
