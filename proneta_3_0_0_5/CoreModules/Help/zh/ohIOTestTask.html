4 IO 测试任务

IO 测试任务用于在安装 CPU 之前检查分布式 I/O 设备的接线，并以快速、准确且简单的方式拟定测试结果。

注意

由于 Microsoft Windows 的限制，PRONETA Basic 在 IO 测试中的通信周期为 256 ms。这意味着，如果安装设备中的信号变化速度更快，则这些变化可能会被忽略，并且可能在“I/O 事件日志”(I/O Event Log) 中丢失，或者间歇性显示。

注意

对于电机起动器单元，仅支持在固件版本 V1.1 及更高版本中进行 IO 测试。

在实际测试之前，必须在“设备选择”模式下选择设备。
